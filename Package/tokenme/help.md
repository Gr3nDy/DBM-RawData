# TokenMe

Convert <b>Member Data</b> into a reedemable <b>Token</b>

<b>Usage</b>
* `[p]token create <datatype> <value> <duration>`
* `[p]token gift <user> <datatype> <value> <duration>`
* `[p]token list`
* `[p]token check <token>`
* `[p]token delete <token>`
* `[p]redeem <token>`
